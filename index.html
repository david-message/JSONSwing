JSONSwing by steowens

SWING Based Editor tools for JSON

This project includes an editor widget and an editor frame which enables the creation and modification of valid JSON using a tree based editor combined with a text based editor.

Screenshot and Overview

The screenshot above shows the JSONEditFrame in action. The JSONEditFrame has the ability to be use (almost as a dialog) or as a full blown main frame. There are 2 rows of buttons on the dialog flavor of the frame. The top row (just below the text area) are for manipulating the JSON object model maintained by the Tree view. You can click on a node in the tree and "Get" the JSON which will replace the contents of the text area. You can edit the contents of the text area, select a node in the tree and use one of the available manipulation buttons to "Replace" a selected node, "Insert" before the selected node, "Append" (e.g. Insert after the selected node), add a "New Child" to a selected node. You can also "Rename" a node which belongs to an Object parent. You can "Delete" the selected node as well.

When you click OK the calling code can get the JSON from the edit frame and do with it what it wishes.

Install

Get the dependencies from the project sites, set up symlinks to them in the project lib directory if you want to build the project. If you just want to use the binary distribution simply ensure that the release jar file for JSONSwing along with it's dependencies are in your classpath. If using the release distribution you don't need Junit or hamcrest.

Using in an App

The above code is what displayed the edit frame in the screenshot above. To use as a "modal" dialog simply implement an OKCancelListener. Then instantiate the frame like this theFrame = new JSONEditFrame(this, "Testing JSONEditFrame", "{}"); The first argument to the constructor provides a reference to the OKCancelListener. This is JSONEditFrame's clue that this frame should be used as a dialog (and provide OK/CANCEL buttons).

The second argument is the frame title, and the last argument is the JSON to initialize the model with.

When the user clicks: OK, CANCEL or the Default Close button the public void onFrameAction(Action action, JFrame frame) method in your listener will be called. You can look at the Action and decide what to do.
